# CS-Programming-Lab-Exercises
C++ lab exercises covering structures, 2D arrays, string manipulation, and pointers.

This repository contains C++ programming exercises.

**Question 1 is about Structures and Arrays. It is a program to manage customer data using an array of structures.**

**Question 2 is about 2D Arrays. It calculates the sum of main and anti-diagonals in a 3x3 matrix.**

**Question 3 is about Strings and Loops. It reverses a string without using built-in library functions.**

**Question 4 is about Pointers and Memory. It demonstrates swapping variable values using pointers.**
