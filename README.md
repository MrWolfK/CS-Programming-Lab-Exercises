# CS-Programming-Lab-Exercises

### This repository contains C++ programming exercises.



Question 1 is about Structures and Arrays. It is a program to manage customer data using an array of structures.

Question 2 is about 2D Arrays. It calculates the sum of main and anti-diagonals in a 3x3 matrix.

Question 3 is about Strings and Loops. It reverses a string without using built-in library functions.

Question 4 is about Pointers and Memory. It demonstrates swapping variable values using pointers.

Question 5 is about Nested IF/ELSE. It finds the largest number among four inputs.

Question 6 is about Sorting with Conditions. It sorts five numbers in ascending order using IF/ELSE IF.

Question 7 is about Conditional Logic. It calculates quarterly water bills. 

Question 8 is about Formatting Output. It prints a formatted student result card.

Question 9 is about the While Loop. It is a fundraising calculator determining contributions based on laps.

Question 10 is about the For Loop and Modulus. It calculates the Greatest Common Divisor (GCD) of three numbers.

Question 11 is about Advanced Looping. It calculates the Least Common Multiple (LCM) of three numbers.

Question 12 is about Algorithms. It reverses a number to check if it is a palindrome and sums its digits.

Question 13 is about Data Aggregation. It calculates the fuel efficiency (Km/L) for individual trips and total overall mileage.

Question 14 is about Switch Cases & Do-While Loops. It acts as a basic bank ATM simulator.

Question 15 is about Business Logic. It calculates new balances for a store account and checks credit limits.

Question 16 is about Hourly Calculations. It calculates accrued leave time for employees based on hours worked.
